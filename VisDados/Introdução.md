As palavras nem sempre nos dão uma visão muito precisa. Por outro lado, dados brutos nem sempre contam a história mais convincente. Para que uma visualização de dados seja boa, é necessário:
#### Entender Contexto
Análise exploratória identifica o tipo de dados e a semântica de cada dado, geralmente realizada em tabela, porém não recomendado para apresentações públicas.
A análise explanatória, cuja finalidade é construir visualizações e explicar os dados seleciona histórias relevantes para serem contadas sobre o dataset, geralmente já passa a utilizar gráficos, sendo recomendada para apresentações.
