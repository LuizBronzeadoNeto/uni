#### Padrão
O que é um **padrão**? É uma abstração para algum objeto concreto, como caracteristicas que descrevem uma assinatura. Pode também ser um objeto abstrato ou uma representação quantitativa ou estrutural de uma entidade ou objeto de interesse.
##### Classes de padrões
Conjunto de Padrões (não necessariamente idênticos) que compartilham propriedades em comum (como descritores [[Image representation and extraction]].
##### Representações de padrões
Seja $x$ um conjunto de dados.
* Classificação: encontrar uma função $f:x \rightarrow C$ em que $C$ é um conjunto discreto
* Regressão: encontrar uma função $f: x \rightarrow y$ em que $y \subseteq \mathbb{R}$ é um espaço contínuo
* Agrupamento: agrupar dados de $x$ em conjuntos $k$
* Estimação de densidade: modelar $p(x)$, que é a densidade das probabilidades de $x$ 
#### Support Vector Machines
É uma extensão do método linear de classificação, envolve a formação de vetores n dimencionais para encontrar a maior distância entre n classes
